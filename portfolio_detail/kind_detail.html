더 착한 커피 웹 디자인 + 퍼블리싱

  • 카테고리: 식품(카페)
  • 기여도: 디자인 100% + 퍼블리싱 100%
  • 사용 기술: HTML + CSS + JQUERY
  • 메인 페이지
  • 서브 페이지(메뉴)
  • 서브 페이지(맴버십)

상세 개요

Color :

  라떼 전문 카페만의 따뜻한 느낌을 연출하기 위해 색 선정을 커피&원두의 느낌으로 가져갔고 제일 중요한 메뉴와 이벤트들을 최상단에 배치하여 사용자의 편의성을 높혔습니다.

  중하단 부분의 회사 인포를 경계로 하여 상단,하단 섹션을 보다 명확하게 하였으며 아래 컨텐츠들에 동적인 요소를 추가하고 플렉스를 이용한 액자 배치로 감각적인 모습을 연출하였습니다.

  서브 페이지는 정보를 쉽게 전달하기 위해 아이콘과 사진을 적절히 이용하며 정적으로 제작하였습니다.
